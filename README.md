# ABOUT PROJECT
## Project Description:
Pixel App is a react web appication for picture search. User can search pictures by tags. It's built with dynamic features, such as pagination, zoom in picture, and light/dark mode.

## Project technologies:
- Build the frontend with React.
- Build the backend with Express.
- Use CSS and Bootstrap for the styling.
- Use Axios library to fatch data.
- Use React Router DOM to implement dynamic routing.
